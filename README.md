## 深度学习项目
### 项目相关
* 前端：bootstrap，jquery
* 后台：django, tensorflow
* 数据集：Mnist
* 网络结构： CNN-Pool-CNN-Pool-CNN-Full Connec-Softmax

----------
### 运行方法
- 新建一个虚拟环境，然后安装相关模块```pip install -r requirements/dev.txt```
- 初始化表```python manage.py migrate```
- 最后运行```python manage.py runserver```
